

# $\sigma$ - Algebra


An for a given set $X$, $(X, \mathcal{A})$ is a $\sigma$ - Algebra if:

- A1.:

$$\mathcal{A} \subset \mathcal{P}(X)$$

- A2. **Closure Under Compliment**:

$$E \in \mathcal{A} \implies X \setminus E \in \mathcal{A}$$

- A3. **Closure Under Countable Unions**

$$S \subset \mathcal{A} \land |S| < \mathfrak{c} \implies \bigcup_{E \in S} E$$
